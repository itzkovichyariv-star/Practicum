הוראות שיתוף

  1. שמרו את קובץ ה-HTML בתיקיית OneDrive משותפת
  2. כל עריכה שומרת גיבוי JSON אוטומטי - הגדירו את תיקיית ההורדות של הדפדפן לתיקייה משותפת
  3. כשפותחים את המערכת, טענו את קובץ הגיבוי האחרון מ"גיבוי" ← "טען גיבוי"

גיבוי אוטומטי

איך זה עובד:
1. כל שינוי נשמר אוטומטית בדפדפן (localStorage)
2. כל שמירה מוריד גיבוי JSON אוטומטי (מקסימום פעם ב-30 שניות)
3. טיפ: שנה את תיקיית ההורדות של Edge לתיקיית המערכת ב-OneDrive:
Edge → הגדרות → הורדות → שנה מיקום ל:
